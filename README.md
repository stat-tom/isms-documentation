# isms-documentation

This repository demonstrates practical implementation of GRC concepts based on ISO 27001 and NIST frameworks.

## Repository Structure

The repository is organized into topic-based folders that mirror a practical ISMS documentation set:

- `00-governance`
- `01-risk-management`
- `02-policies`
- `03-procedures`
- `04-controls`
- `05-audit`
- `06-compliance`
- `07-assets`
- `08-suppliers`
- `09-awareness`
- `10-continual-improvement`
- `evidence`
